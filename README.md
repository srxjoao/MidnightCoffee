## 🗂 Executar Projeto

```bash
# Clonar Projeto.
$ git clone https://github.com/srxjoao/MidnightCoffee.git

# Entrar na pasta servidor.
$ cd servidor

# Baixar as dependencias.
$ npm install

# Executar.
$ npm start

# Novo cmd.
# Entrar na pasta cliente.
$ cd cliente

# Baixar as dependencias.
$ npm install

# Executar.
$ npm run dev

 ```
